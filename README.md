# 📸 Instagram Web Layout Clone

A high-fidelity desktop frontend replica of the Instagram web client, meticulously recreating the user interface, sidebar navigation, story banners, post feed cards, and responsive side overlays.

## 🛠️ Features

*   📌 **Authentic Sidebar Nav**: Fixed desktop navigation menu featuring pixel-perfect replicas of official Instagram SVG icons (Home, Search, Explore, Reels, Messages, Notifications, Create, Profile).
*   ⭕ **Gradient Stories Bar**: Scrollable story banners decorated with the trademark red-orange gradient rings.
*   📰 **High-Fidelity Post Feed**: Detailed post cards with profile headers, location tags, follow toggles, media containers, interaction buttons, likes counter, captions, and comments.
*   📱 **Responsive Flexbox/Grid**: Adapts fluidly to desktop screen adjustments, maintaining clean proportions.

---

## 💻 Tech Stack

*   **Structure**: HTML5 (`index.html`)
*   **Styling**: CSS3 (`style.css`)
    *   *Layout*: Flexbox containers for feed centering and sidebar grid structures.
    *   *Assets*: Dynamic vector SVGs for crisp UI icons.
*   **Media**: Local videos and images loaded dynamically.

---

## 🏃 How to Run

Test the layout inside your browser:

```
[1] Navigate to the instagram-clone directory.
[2] Double-click index.html to launch.
 ↳ 📱 Instagram Desktop layout loaded in your browser window!
[3] Hover and interact with the sidebar buttons and scroll through the feed.
```
